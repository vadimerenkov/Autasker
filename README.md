Autasker is a free and open-source to-do list application. It's a simple but robust task manager designed to be as comfortable to use as possible and neurodivergent-friendly.

Highlights:

- Create tasks and sort them by categories and tabs
- The day ends when you say it ends - just change the time in the settings!
- A powerful system of repeating tasks - fine-tune the way each task repeats individually
- No yelling at you for missing or skipping the tasks - you control the app, not the other way around
- A smart system for planning your day every morning: just open the app and it will suggest some tasks to do today
- Supported on Windows, Linux and Android
- Fully offline - no need to create an account, your data stays with you
- In active development: new features soon to come

Powered by Kotlin Multiplatform and Jetpack Compose.
